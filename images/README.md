# Images Folder

Place your site images here:

- `favicon.png` - Browser tab icon (recommended: 32x32 or 64x64 pixels)
- `logo.png` - Site logo (optional, if you want an image logo)
- Any other site-wide images

## Favicon

To create a favicon:
1. Create a square image (at least 64x64 pixels)
2. Save as `favicon.png` or `favicon.ico`
3. The HTML files already reference `images/favicon.png`

## Logo

If you want to use an image logo instead of text:
1. Add your logo image here
2. Edit the `.logo` section in each HTML file
