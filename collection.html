This gallery automatically displays images from your collection folder. Here's how to add your images:

Option 1: Using a JSON Manifest (Recommended)

Create a file called collection/images.json with this format:

{
  "images": [
    {
      "filename": "coin-1921-morgan-dollar.jpg",
      "title": "1921 Morgan Silver Dollar",
      "category": "coins",
      "description": "Uncirculated condition"
    },
    {
      "filename": "book-hemingway-first-edition.jpg",
      "title": "Hemingway First Edition",
      "category": "books"
    }
  ]
}

Option 2: Using GitHub API (Auto-detection)

Edit js/collection.js and set:

const CONFIG = {
  USE_GITHUB_API: true,
  GITHUB_REPO: 'your-username/haq-collectibles',
  // ... rest of config
};
